# Amanecer Gaming | eSports Portal

![Amanecer Gaming Header](hero_bg.png)

Bienvenido al repositorio oficial del portal web de **Amanecer Gaming**, una organización líder en eSports centrada en elevar el nivel competitivo de Latinoamérica Sur.

## 🚀 Características

- **Gestión de Torneos:** Visualización de brackets competitivos, tablas de posiciones (ligas) y estados de torneos en tiempo real.
- **Sincronización Global (Nexus Engine):** Integración con Firebase Firestore para sincronización instantánea de datos y estadísticas sociales.
- **Sistema Multilingüe:** Soporte completo para Español y Portugués.
- **Perfiles de Jugador:** Sistema de registro, inicio de sesión (simulado con verificación de correo) y paneles de estadísticas de Mobile Legends.
- **Diseño Premium:** Interfaz oscura, moderna y responsiva con estética inspirada en los principales títulos de eSports.
- **Portal Administrativo:** Panel interno para que el staff gestione rostros, premios y estados de torneos.

## 🛠️ Tecnologías

- **Frontend:** HTML5, CSS3 (Vanilla), JavaScript (ES6+).
- **Base de Datos / Backend:** Firebase Firestore (Cloud Engine).
- **Tipografía:** Google Fonts (Outfit).
- **Iconografía:** Font Awesome 6.

## 📁 Estructura del Proyecto

- `index.html`: Estructura principal y contenedores del portal.
- `style.css`: Sistema de diseño con variables CSS, animaciones y media queries para móviles.
- `app.js`: Motor lógico de la aplicación, manejo de estados, navegación y sincronización con la nube.
- `data.js`: Base de datos local inicial y estructura de objetos de la organización.
- `translations.js`: Diccionarios para el sistema multi-idioma.
- `hero_bg.png`: Fondo artístico del encabezado generado a medida.

## 💻 Desarrollo Local

1. Clona este repositorio:
   ```bash
   git clone https://github.com/TU_USUARIO/amanecer-gaming-portal.git
   ```
2. Abre el archivo `index.html` en cualquier navegador moderno.
3. Para la sincronización en tiempo real, asegúrate de configurar las reglas de seguridad en tu consola de Firebase Firestore.
